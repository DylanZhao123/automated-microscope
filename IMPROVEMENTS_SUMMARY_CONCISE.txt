ENGLISH VERSION
───────────────────────────────────────────────────────────────────────────

Bug Fixes
─────────
• Fixed missing Flake object properties that caused crashes
• Removed all hardcoded file paths - created centralized config.py
• Added proper error handling for files, serial port, and JSON parsing
• Made parameter loader compatible with multiple JSON formats


Detection Improvements
──────────────────────
• Lowered confidence threshold from 0.75 to 0.5 for better coverage
• Removed "one flake per layer" restriction to match actual output
• Added multi-layer support (1L-5L, currently using 1L-2L)
• Enabled flatfield correction for better detection on real images


Visualization Fixes
───────────────────
• Changed output to show only the best flake (not all detected)
• Switched from rainbow colors to magenta outline (matching reference)
• Fixed text format to exact specification: "1. 2L 304um2 84%"
• Made connector line start from text end instead of fixed position


New Features
────────────
• Created test_detection.py for single image testing
• Created batch_test scripts for processing multiple images
• Set default output to new_outputs/ folder for organization
• Added parameter format conversion tool (convert_params.py)
• Wrote comprehensive documentation of changes and usage


Code Quality
────────────
• Fixed Python import paths so scripts run from correct directories
• Prioritized standard Graphene_GMM.json over old parameter format
• Made all output paths configurable with sensible defaults


RESULT: System now runs reliably, matches reference output format exactly,
        and is much easier to use and maintain.
